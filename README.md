# ![image](https://github.com/user-attachments/assets/d524c3e1-27e2-4d5b-8d21-5c61d898e910) Simple Password Generator

A lightweight and intuitive password generator built with Python's native Tkinter library. This tool is designed for anyone who needs to create strong, secure passwords quickly and without hassle.

## Features

- **Flexible Generation:** Specify the exact length for your password, from a short PIN to a long, complex passphrase.
- **Complexity Control:** Easily toggle digits and special symbols to meet varying security requirements.
- **Sleek Interface:** A clean, straightforward GUI that includes a dark theme for comfortable use in any lighting.
- **Instant Copy:** The generated password can be copied to your clipboard with a single click, streamlining your workflow.

## Getting Started

### Prerequisites
- Python 3.x

### Installation & Running
1.  **Clone the repository:**
    ```bash
    git clone https://github.com/LeLeLeonid/password-generator.git
    cd simple-password-generator
    ```
2.  **Install the required dependencies:**
    ```bash
    pip install -r requirements.txt
    ```
3.  **Launch the application:**
    ```bash
    python main.py
    ```

## Dependencies

This project relies on the following packages:
- `pyperclip`: For clipboard functionality.
- `Tkinter`: As part of the standard Python library.
